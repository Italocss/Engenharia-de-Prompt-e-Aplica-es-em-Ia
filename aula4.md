# 🧠 Projeto de Engenharia de Prompt e Aplicações de IA

## 📖 Descrição
Este projeto explora técnicas avançadas de **engenharia de prompt** aplicadas a modelos de Inteligência Artificial.  
O objetivo é demonstrar como diferentes estratégias de construção de prompts podem influenciar a qualidade, relevância e criatividade das respostas geradas por sistemas de IA.

Principais pontos abordados:
- Estruturação de prompts para tarefas específicas
- Comparação entre abordagens diretas e contextuais
- Aplicações práticas em geração de texto, suporte a decisão e automação

💡 Exemplo
A seguir, um exemplo simples de uso do projeto para gerar respostas de IA com diferentes estilos de prompt:

python
from prompt_engineering import PromptEngine

engine = PromptEngine(model="gpt-4")

# Prompt direto
resposta1 = engine.run("Explique o conceito de aprendizado de máquina em 3 frases.")

# Prompt contextual
resposta2 = engine.run("""
Você é um professor de ciência da computação.
Explique para iniciantes o que é aprendizado de máquina,
usando uma linguagem simples e exemplos do dia a dia.
""")

print("Resposta direta:", resposta1)
print("Resposta contextual:", resposta2)
